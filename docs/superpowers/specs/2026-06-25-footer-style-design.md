# 里程碑4 协议/数据来源 样式 设计（note 31）

## 背景

- 协议链接用品牌红 `#e53935`，太显眼（note 31：协议颜色淡一点不要太明显）。

## 方案

不动 `legal.js`（协议正文里对数据来源的说明属协议条款，保留）。其它页面的来源展示（海报）已在里程碑1 改为纯网址。
